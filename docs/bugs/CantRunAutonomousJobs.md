# CantRunAutonomousJobs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make natural-language Telegram reminder requests create real scheduled `autonomous_jobs` rows so the unified scheduler can run them later, instead of sending the reminder immediately.

**Architecture:** Reuse the existing `autonomous_jobs` table, `AutonomousJobService`, and unified scheduler. Add the missing model-callable local tool `tdai_create_job` so chat mode can persist hybrid scheduled jobs; due jobs send fixed Telegram text first, then run the agent prompt, increment run count, and delete themselves when their run limit is reached. Do not edit `dist/index.js` manually; rebuild it from TypeScript before running `bun run start`.

**Tech Stack:** Bun, TypeScript, grammY, bun:sqlite, node-cron, cron-parser, Bun test.

---

## Current diagnosis

The scheduler is running, but it has no autonomous job to run.

Observed evidence:

- `package.json:8-9` shows `bun run start` runs `bun dist/index.js`, not `src/index.ts`. Changes in `src/` have no runtime effect until `bun run build` is executed and the running process is restarted.
- `dist/index.js` currently contains `telegram_send_message`, but no `tdai_create_job` tool.
- Runtime `data/agent.db` has an empty `autonomous_jobs` table.
- Runtime `data/agent.db` has `memory_task_nodes` rows for `telegram_send_message`; those are tool-call history records, not scheduler jobs.
- `src/cron/scheduler.ts:23-24` only dispatches jobs returned by `AutonomousJobService.listDueJobs()`.
- `src/services/autonomous-jobs.ts:268-300` only lists due rows from `autonomous_jobs`.
- `src/tools/local.ts:144-164` exposes `telegram_send_message`, but no local tool that creates scheduled jobs from chat.

Root cause:

1. The LLM handled “ingatkan saya dalam 5 menit...” by calling `telegram_send_message` immediately.
2. No `autonomous_jobs` row was created.
3. The scheduler tick correctly logged `jobsRun: 0` because the scheduler only sees rows in `autonomous_jobs`.
4. The running app is using stale `dist/index.js`, so source changes made after the last build are not active.

```ts
function asRecord(value: unknown): Record<string, unknown> {
  return value && typeof value === "object" && !Array.isArray(value) ? (value as Record<string, unknown>) : {};
}

function asPositiveInteger(value: unknown): number | null {
  return typeof value === "number" && Number.isInteger(value) && value > 0 ? value : null;
}

function defaultMaxRuns(value: unknown): number | string {
  if (value === undefined || value === null) return 1;
  return asPositiveInteger(value) ?? "max_runs must be a positive integer.";
}

function parseRunAtUnix(value: unknown): number | string {
  const raw = asString(value).trim();
  if (!raw) return "schedule.run_at is required for one-shot jobs.";

  const millis = Date.parse(raw);
  if (!Number.isFinite(millis)) return "schedule.run_at must be a valid date/time string.";

  return Math.floor(millis / 1000);
}
```

In `src/tools/local.ts`, add this registered tool object immediately before the existing `telegram_send_message` tool:

```ts
{
  name: "tdai_create_job",
  source: "local",
  description:
    "Create a scheduled autonomous Telegram job for reminders or recurring tasks. Use tdai_current_datetime first for relative time requests, then pass an absolute run_at date/time for one-shot jobs.",
  inputSchema: {
    type: "object",
    properties: {
      message_text: { type: "string", description: "Fixed Telegram text sent when the job is due" },
      agent_prompt: { type: "string", description: "Prompt run by the agent after the fixed text is sent" },
      schedule: {
        type: "object",
        properties: {
          mode: { type: "string", enum: ["once", "interval", "cron"] },
          run_at: { type: "string", description: "Absolute date/time string for one-shot jobs, preferably ISO with timezone" },
          interval_sec: { type: "number", description: "Positive integer seconds for interval jobs" },
          cron_expr: { type: "string", description: "Cron expression for cron jobs" },
        },
        required: ["mode"],
        additionalProperties: false,
      },
      max_runs: { type: "number", description: "Positive integer run limit; defaults to 1" },
    },
    required: ["message_text", "agent_prompt", "schedule"],
    additionalProperties: false,
  },
  async execute(args, ctx) {
    if (!autonomousJobs) return "Autonomous job service unavailable.";

    const messageText = asString(args.message_text).trim();
    if (!messageText) return "message_text is required.";

    const agentPrompt = asString(args.agent_prompt).trim();
    if (!agentPrompt) return "agent_prompt is required.";

    const maxRuns = defaultMaxRuns(args.max_runs);
    if (typeof maxRuns === "string") return maxRuns;

    const scheduleInput = asRecord(args.schedule);
    const mode = asString(scheduleInput.mode).trim();

    if (mode === "once") {
      const runAtUnix = parseRunAtUnix(scheduleInput.run_at);
      if (typeof runAtUnix === "string") return runAtUnix;

      const job = autonomousJobs.createJob({
        chatId: ctx.chatId,
        userId: ctx.userId,
        prompt: agentPrompt,
        jobType: "hybrid",
        messageText,
        agentPrompt,
        schedule: { scheduleMode: "once", runAtUnix },
        maxRuns,
      });
      return `Created autonomous job #${job.id}. Schedule: ${job.scheduleLabel}. Max runs: ${job.maxRuns ?? "unlimited"}.`;
    }

    if (mode === "interval") {
      const intervalSec = asPositiveInteger(scheduleInput.interval_sec);
      if (intervalSec === null) return "schedule.interval_sec must be a positive integer.";

      const job = autonomousJobs.createJob({
        chatId: ctx.chatId,
        userId: ctx.userId,
        prompt: agentPrompt,
        jobType: "hybrid",
        messageText,
        agentPrompt,
        schedule: { scheduleMode: "interval", intervalSec },
        maxRuns,
      });
      return `Created autonomous job #${job.id}. Schedule: ${job.scheduleLabel}. Max runs: ${job.maxRuns ?? "unlimited"}.`;
    }

    if (mode === "cron") {
      let cronExpr = "";
      try {
        cronExpr = validateCronExpression(asString(scheduleInput.cron_expr));
      } catch (error) {
        return error instanceof Error ? error.message : String(error);
      }

      const job = autonomousJobs.createJob({
        chatId: ctx.chatId,
        userId: ctx.userId,
        prompt: agentPrompt,
        jobType: "hybrid",
        messageText,
        agentPrompt,
        schedule: { scheduleMode: "cron", cronExpr },
        maxRuns,
      });
      return `Created autonomous job #${job.id}. Schedule: ${job.scheduleLabel}. Max runs: ${job.maxRuns ?? "unlimited"}.`;
    }

    return "schedule.mode must be one of: once, interval, cron.";
  },
},
```
